# 🧮 Calculadora de Lógica Digital (Simplificador Lógico)

Uma aplicação web interativa e responsiva desenvolvida para auxiliar no estudo e resolução de problemas de Lógica Digital e Álgebra Booleana. O sistema automatiza a criação de Tabelas Verdade, Mapas de Karnaugh e a simplificação de expressões lógicas passo a passo.

Projeto desenvolvido como material de apoio para a UC de **Matemática Computacional Aplicada - Ânima**.

## ✨ Funcionalidades

* **Análise de Expressões:** Avaliação de expressões booleanas digitadas pelo usuário com suporte a múltiplos operadores (AND, OR, NOT, XOR).
* **Tabela Verdade Dinâmica:** Geração automática da tabela para sistemas de 2, 3 ou 4 variáveis.
* **Mapa de Karnaugh Interativo:** Visualização em grade seguindo o código de Gray, permitindo que o usuário clique nas células para montar sua própria lógica.
* **Simplificação Algébrica:** Implementação do método de Quine-McCluskey para encontrar a expressão mínima (Implicantes Primos), exibindo o passo a passo do cálculo.
* **Exemplos Rápidos:** Biblioteca de exercícios pré-carregados para testar o sistema instantaneamente.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído no modelo *Vanilla* (sem frameworks JavaScript complexos) para garantir máxima velocidade e simplicidade de execução:

* **HTML5 & CSS3:** Estrutura semântica.
* **JavaScript (Vanilla):** Lógica de cálculo, algoritmo de Quine-McCluskey e manipulação do DOM.
* **Tailwind CSS (via CDN):** Estilização rápida, moderna e responsiva.
* **FontAwesome:** Ícones da interface (UI/UX).
